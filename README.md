10. ホモロジーの計算 2
11. テンソルの定義
12. テンソル解析 1
13. テンソル解析 2

# コンパイル
`uplatex`と`dvipdfmx`が必要です．ターミナル上で

```
git clone https://github.com/tobetakoyaki/GeometricalMathematics.git
cd GeometricalMathematics
make
```
を実行することで，`resume.pdf`が生成されます．
